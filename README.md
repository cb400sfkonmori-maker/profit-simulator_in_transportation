## 📌 プロジェクトの使命

日本のインフラを支える運輸・物流業界は今、燃料高騰、2024年問題、そして深刻な人手不足という三重苦に直面しています。
かつて国内トップシェアのインフラ資材メーカーで14.5年、品質保証（QA）と原価管理の最前線に立ってきた私は、多くの現場で**「手段（会計）が目的（利益）を食いつぶし、現場が劣化していく姿」**を目撃してきました。

このシミュレーターは、経営者が気づきにくい「現場に潜む利益の漏れ」を可視化し、再び「誇りを持って稼げる現場」を取り戻すための第一歩として開発されました。

---

## 📊 診断できる「2つの損失」

本ツールでは、運輸業において最も「劣化」が数字に表れやすい2項目をシミュレーションします。

### 1. 待機時間の損失 ($L_w$)
荷主都合による待機時間は、単なる時間の浪費ではなく、車両固定費と人件費の流出です。
$$L_w = H_w \times 2,500 \times 260 \times N$$
*(待機時間 $H_w$, 稼働日数 260日, 車両台数 $N$)*

### 2. 燃費劣化の損失 ($G_f$)
現場の規律や整備状況のわずかな差が、年間数百万のコスト増を招きます。
$$G_f = \left( \frac{M}{E_{curr}} - \frac{M}{E_{target}} \right) \times 160 \times N$$
*(年間走行距離 $M$, 現在燃費 $E_{curr}$, 目標燃費 $E_{target}$, 車両台数 $N$)*

## 🚀 特徴

- **直感的なUI**: 経営者のスマホ（Pixel 10等）で、スライダーを動かすだけで瞬時に損失額を算出。
- **現場主義のロジック**: 理論値ではなく、14.5年の実務経験に基づいた「実効性のある係数」を採用。
- **GitHub Pages対応**: インストール不要。QRコード一つで、商談のその場で診断可能。
